description: >
  <p>Creates a new <a href="#/p5.Shader">p5.Shader</a> using only a fragment
  shader, as a convenience method for creating image effects.

  It's like <a href="#/createShader">createShader()</a> but with a default
  vertex shader included.</p>

  <p><a href="#/createFilterShader">createFilterShader()</a> is intended to be
  used along with <a href="#/filter">filter()</a> for filtering the contents of
  a canvas in WebGL mode.

  A filter shader will not be applied to any geometries.</p>

  <p>The fragment shader receives some uniforms:</p>

  <ul>

  <li><code>sampler2D tex0</code>, which contains the canvas contents as a
  texture</li>

  <li><code>vec2 canvasSize</code>, which is the width and height of the
  canvas</li>

  <li><code>vec2 texelSize</code>, which is the size of a pixel
  (<code>1.0/width</code>, <code>1.0/height</code>)</li>

  </ul>

  <p>For more info about filters and shaders, see Adam Ferriss' <a
  href="https://github.com/aferriss/p5jsShaderExamples">repo of shader
  examples</a>

  or the <a
  href="https://p5js.org/learn/getting-started-in-webgl-shaders.html">introduction
  to shaders</a> page.</p>

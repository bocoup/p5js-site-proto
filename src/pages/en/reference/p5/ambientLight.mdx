description: |
  <p>Creates an ambient light with the given color.</p>
  <p>Ambient light does not come from a specific direction.
  Objects are evenly lit from all sides. Ambient lights are
  almost always used in combination with other types of lights.</p>
  <p>Note: lights need to be called (whether directly or indirectly)
  within draw() to remain persistent in a looping program.
  Placing them in setup() will cause them to only have an effect
  the first time through the loop.</p>

  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100, WEBGL);
      noStroke();
      describe('sphere with coral color under black light');
    }
    function draw() {
      background(100);
      ambientLight(0); // black light (no light)
      ambientMaterial(255, 127, 80); // coral material
      sphere(40);
    }
    </code>

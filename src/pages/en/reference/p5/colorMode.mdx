description: >
  <p>Changes the way p5.js interprets color data. By default, the numeric

  parameters for <a href="#/p5/fill">fill()</a>,

  <a href="#/p5/stroke">stroke()</a>,

  <a href="#/p5/background">background()</a>, and

  <a href="#/p5/color">color()</a> are defined by values between 0 and 255

  using the RGB color model. This is equivalent to calling

  <code>colorMode(RGB, 255)</code>. Pure red is <code>color(255, 0, 0)</code> in
  this model.</p>

  <p>Calling <code>colorMode(RGB, 100)</code> sets colors to be interpreted as
  RGB color

  values between 0 and 100.  Pure red is <code>color(100, 0, 0)</code> in this
  model.</p>

  <p>Calling <code>colorMode(HSB)</code> or <code>colorMode(HSL)</code> changes
  to HSB or HSL system

  instead of RGB.</p>

  <p><a href="#/p5.Color">p5.Color</a> objects remember the mode that they were

  created in. Changing modes doesn't affect their appearance.</p>

    noStroke();

    colorMode(RGB, 100);

    for (let i = 0; i < 100; i += 1) {
      for (let j = 0; j < 100; j += 1) {
        stroke(i, j, 0);
        point(i, j);
      }

description: >
  <p>Sets the style for system fonts when

  <a href="#/p5/text">text()</a> is called. <code>textStyle()</code> accepts the

  following values: <code>NORMAL</code>, <code>ITALIC</code>, <code>BOLD</code>
  or <code>BOLDITALIC</code>.</p>

  <p><code>textStyle()</code> may be overridden by CSS styling. This function
  doesn't

  affect fonts loaded with <a href="#/p5/loadFont">loadFont()</a>.</p>
line: 152
itemtype: method
class: p5
example:
  - >-

    <div>

    <code>

    textSize(12);

    textAlign(CENTER);


    textStyle(NORMAL);

    text('Normal', 50, 15);

    textStyle(ITALIC);

    text('Italic', 50, 40);

    textStyle(BOLD);

    text('Bold', 50, 65);

    textStyle(BOLDITALIC);

    text('Bold Italic', 50, 90);


    describe('The words "Normal" displayed normally, "Italic" in italic, "Bold"
    in bold, and "Bold Italic" in bold italics.');

    </code>

    </div>

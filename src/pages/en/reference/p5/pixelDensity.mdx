  <p>Sets the pixel scaling for high pixel density displays.</p>

  <p>By default, the pixel density is set to match display density. Calling

  <code>pixelDensity(1)</code> turn this off.</p>

  <p>Calling <code>pixelDensity()</code> without an argument returns the current
  pixel

  density.</p>

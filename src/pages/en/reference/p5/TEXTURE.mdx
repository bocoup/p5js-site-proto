description: >
  <p>Sets the texture that will be used to render subsequent shapes.</p>

  <p>A texture is like a "skin" that wraps around a 3D geometry. Currently

  supported textures are images, video, and offscreen renders.</p>

  <p>To texture a geometry created with <a
  href="#/p5/beginShape">beginShape()</a>,

  you will need to specify uv coordinates in <a
  href="#/p5/vertex">vertex()</a>.</p>

  <p>Note, texture() can only be used in WEBGL mode.</p>

  <p>You can view more materials in this

  <a href="https://p5js.org/examples/3d-materials.html">example</a>.</p>

  <p>Sets the seed value for <a href="#/p5/noise">noise()</a>. By default,

  <a href="#/p5/noise">noise()</a> produces different results each time

  a sketch is run. Calling <code>noiseSeed()</code> with a constant

  argument, such as <code>noiseSeed(99)</code>, makes <a
  href="#/p5/noise">noise()</a>

  produce the same results each time a sketch is run.</p>

      // Scale input coordinate.
      let nt = noiseScale * frameCount;
      // Compute noise value.
      let x = noiseLevel * noise(nt);
      // Render.
      line(x, 0, x, height);

  <p>The <a href="#/p5/mouseDragged">mouseDragged()</a> function is called once
  every time the mouse moves and

  a mouse button is pressed. If no <a
  href="#/p5/mouseDragged">mouseDragged()</a> function is defined, the

  <a href="#/p5/touchMoved">touchMoved()</a> function will be called instead if
  it is defined.<br><br>

  Browsers may have different default

  behaviors attached to various mouse events. To prevent any default

  behavior for this event, add "return false" to the end of the function.</p>

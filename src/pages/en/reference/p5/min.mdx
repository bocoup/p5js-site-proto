  <p>Returns the smallest value in a sequence of numbers.</p>

  <p>The version of <code>min()</code> with one parameter interprets it as an
  array of

  numbers and returns the smallest number.</p>

  <p>The version of <code>min()</code> with two or more parameters interprets
  them as

  individual numbers and returns the smallest number.</p>

    numbers.forEach((n, index) => {
      let x = index * spacing;
      let y = 25;
      text(n, x, y);

    // Draw the minimum value in the array.

    let m = min(numbers);

  Draw a cone with given radius and heightDetailX and detailY determine the
  number of subdivisions in the x-dimension and

  the y-dimension of a cone. More subdivisions make the cone seem smoother. The

  recommended maximum value for detailX is 24. Using a value greater than 24

  may cause a warning or slow down the browser.

  - name: height

    type: Number
  - name: detailX
    description: |-
      number of segments,
      the more segments the smoother geometry
      default is 24
    type: Integer
  - name: detailY
    description: |-
      number of segments,
      the more segments the smoother geometry
      default is 1
    type: Integer
  - name: cap
    description: whether to draw the base of the cone
    type: Boolean

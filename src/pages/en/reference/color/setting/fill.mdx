descriptionText: >
  <p>Sets the color used to fill shapes. Calling <code>fill(255, 165, 0)</code>
  or

  <code>fill('orange')</code> means all shapes drawn after the fill command will
  be

  filled with the color orange.</p>

  <p>The version of <code>fill()</code> with one parameter interprets the value
  one of

  three ways. If the parameter is a number, it's interpreted as a grayscale

  value. If the parameter is a string, it's interpreted as a CSS color

  string. A <a href="#/p5.Color">p5.Color</a> object can also be provided to

  set the fill color.</p>

  <p>The version of <code>fill()</code> with three parameters interprets them as
  RGB, HSB,

  or HSL colors, depending on the current

  <a href="#/p5/colorMode">colorMode()</a>. The default color space is RGB,

  with each value in the range from 0 to 255.</p>

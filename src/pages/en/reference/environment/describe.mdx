descriptionText: >-
  Creates a screen reader-accessible description for the canvas.The first
  parameter, `text`, is the description of the canvas.The second parameter,
  `display`, is optional. It determines how the

  description is displayed. If `LABEL` is passed, as in

  `describe('A description.', LABEL)`, the description will be visible in

  a div element next to the canvas. If `FALLBACK` is passed, as in

  `describe('A description.', FALLBACK)`, the description will only be

  visible to screen readers. This is the default mode.Read

  <a href="/learn/labeling-canvases.html">How to label your p5.js code</a> to

  learn more about making sketches accessible.
params:
  - name: text
    description: description of the canvas.
    type: String
  - name: display
    description: either LABEL or FALLBACK.
    type: Constant

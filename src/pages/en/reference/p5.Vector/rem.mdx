  <p>Performs modulo (remainder) division with a vector's <code>x</code>,
  <code>y</code>, and <code>z</code>

  components using separate numbers, another

  <a href="#/p5.Vector">p5.Vector</a> object, or an array of numbers.</p>

  <p>The static version of <code>rem()</code> as in <code>p5.Vector.rem(v2,
  v1)</code>, returns a new

  <a href="#/p5.Vector">p5.Vector</a> object and doesn't change the

  originals.</p>

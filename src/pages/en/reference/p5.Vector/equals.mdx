description: >
  <p>Returns <code>true</code> if the vector's components are all the same as
  another

  vector's and <code>false</code> if not.</p>

  <p>The version of <code>equals()</code> with one parameter interprets it as
  another

  <a href="#/p5.Vector">p5.Vector</a> object.</p>

  <p>The version of <code>equals()</code> with multiple parameters interprets
  them as the

  components of another vector. Any missing parameters are assigned the value

  0.</p>

  <p>The static version of <code>equals()</code>, as in
  <code>p5.Vector.equals(v0, v1)</code>,

  interprets both parameters as <a href="#/p5.Vector">p5.Vector</a> objects.</p>

    let v0 = createVector(10, 20, 30);

    let v2 = createVector(0, 0, 0);

    // Prints "true" to the console.
    print(v0.equals(v1));
    // Prints "false" to the console.
    print(v0.equals(v2));

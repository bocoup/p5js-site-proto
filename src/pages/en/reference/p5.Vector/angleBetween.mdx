description: >
  <p>Returns the angle between two vectors. The angles returned are signed,

  which means that <code>v1.angleBetween(v2) ===
  -v2.angleBetween(v1)</code>.</p>

  <p>If the vector was created with

  <a href="#/p5/createVector">createVector()</a>, <code>angleBetween()</code>
  returns

  angles in the units of the current

  <a href="#/p5/angleMode">angleMode()</a>.</p>

descriptionText: >-
  Sets the color of a pixel or draws an image to the canvas.`set()` is easy to
  use but it's not as fast as

  <a href="#/p5/pixels">pixels</a>. Use <a href="#/p5/pixels">pixels</a>

  to set many pixel values.`set()` interprets the first two parameters as x- and
  y-coordinates. It

  interprets the last parameter as a grayscale value, a `[R, G, B, A]` pixel

  array, a <a href="#/p5.Color">p5.Color</a> object, or a

  <a href="#/p5.Image">p5.Image</a> object. If an image is passed, the first

  two parameters set the coordinates for the image's upper-left corner,

  regardless of the current <a href="#/p5/imageMode">imageMode()</a>.<a
  href="#/p5/updatePixels">updatePixels()</a> must be called after using

  `set()` for changes to appear.
params:

    description: x-coordinate of the pixel.
    type: Number
  - name: 'y'
    description: y-coordinate of the pixel.
    type: Number

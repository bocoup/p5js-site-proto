descriptionText: >-
  Sets the color of one or more pixels within a

  <a href="#/p5.Image">p5.Image</a> object.`img.set()` is easy to use but it's
  not as fast as

  <a href="#/p5.Image/pixels">img.pixels</a>. Use

  <a href="#/p5.Image/pixels">img.pixels</a> to set many pixel
  values.`img.set()` interprets the first two parameters as x- and
  y-coordinates. It

  interprets the last parameter as a grayscale value, a `[R, G, B, A]` pixel

  array, a <a href="#/p5.Color">p5.Color</a> object, or another

  <a href="#/p5.Image">p5.Image</a> object.<a
  href="#/p5.Image/updatePixels">img.updatePixels()</a> must be called

  after using `img.set()` for changes to appear.
params:
  - name: x
    description: x-coordinate of the pixel.
    type: Number
  - name: 'y'
    description: y-coordinate of the pixel.
    type: Number

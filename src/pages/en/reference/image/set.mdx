descriptionText: >
  <p>Sets the color of one or more pixels within a

  <a href="#/p5.Image">p5.Image</a> object.</p>

  <p><code>img.set()</code> is easy to use but it's not as fast as

  <a href="#/p5.Image/pixels">img.pixels</a>. Use

  <a href="#/p5.Image/pixels">img.pixels</a> to set many pixel values.</p>

  <p><code>img.set()</code> interprets the first two parameters as x- and
  y-coordinates. It

  interprets the last parameter as a grayscale value, a <code>[R, G, B,
  A]</code> pixel

  array, a <a href="#/p5.Color">p5.Color</a> object, or another

  <a href="#/p5.Image">p5.Image</a> object.</p>

  <p><a href="#/p5.Image/updatePixels">img.updatePixels()</a> must be called

  after using <code>img.set()</code> for changes to appear.</p>

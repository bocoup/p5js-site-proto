descriptionText: >-
  The touchStarted() function is called once after every time a touch is

  registered. If no <a href="#/p5/touchStarted">touchStarted()</a> function is
  defined, the <a href="#/p5/mousePressed">mousePressed()</a>

  function will be called instead if it is defined.<br><br>

  Browsers may have different default behaviors attached to various touch

  events. To prevent any default behavior for this event, add "return false"

  to the end of the method.
params:
  - name: event
    description: optional TouchEvent callback argument.
    type: TouchEvent

    // Touch within the image to change

    let value = 0;
    function draw() {
      fill(value);
      rect(25, 25, 50, 50);
      describe('50-by-50 black rect turns white with touch event.');
    }
    function touchStarted() {
      if (value === 0) {
        value = 255;
      } else {
        value = 0;
      }
    }
    </code>

descriptionText: >
  <p>Searches the page for all elements that matches the given

  <a
  href="https://developer.mozilla.org/en-US/docs/Learn/Getting_started_with_the_web/CSS_basics#different_types_of_selectors"
  target="_blank">CSS selector string</a>.

  The string can be an ID, class, tag name, or a combination.
  <code>selectAll()</code>

  returns an array of <a href="#/p5.Element">p5.Element</a> objects if it

  finds any matches and an empty array otherwise.</p>

  <p>The second parameter, <code>container</code>, is optional. It specifies a
  container to

  search within. container can be CSS selector string, a

  <a href="#/p5.Element">p5.Element</a> object, or an

  <a href="https://developer.mozilla.org/en-US/docs/Web/API/HTMLElement"
  target="_blank">HTMLElement</a> object.</p>

descriptionText: >
  <p>Schedules a function to call when the audio/video reaches a specific time

  during its playback.</p>

  <p>The first parameter, <code>time</code>, is the time, in seconds, when the
  function

  should run. This value is passed to <code>callback</code> as its first
  argument.</p>

  <p>The second parameter, <code>callback</code>, is the function to call at the
  specified

  cue time.</p>

  <p>The third parameter, <code>value</code>, is optional and can be any type of
  value.

  <code>value</code> is passed to <code>callback</code>.</p>

  <p>Calling <code>media.addCue()</code> returns an ID as a string. This is
  useful for

  removing the cue later.</p>

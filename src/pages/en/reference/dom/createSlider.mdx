  <p>Creates a slider <code>&lt;input&gt;&lt;/input&gt;</code> element. Range
  sliders are

  useful for quickly selecting numbers from a given range.</p>

  <p>The first two parameters, <code>min</code> and <code>max</code>, are
  numbers that set the

  slider's minimum and maximum.</p>

  <p>The third parameter, <code>value</code>, is optional. It's a number that
  sets the

  slider's default value.</p>

  <p>The fourth parameter, <code>step</code>, is also optional. It's a number
  that sets the

  spacing between each value in the slider's range. Setting <code>step</code> to
  0

  allows the slider to move smoothly from <code>min</code> to
  <code>max</code>.</p>

description: >
  <p>Saves the <a href="#/p5.Image">p5.Image</a> object to a file.

  The browser will either save the file immediately or prompt the user

  with a dialogue window.</p>

  <p>By default, calling <code>img.save()</code> will save the image as
  <code>untitled.png</code>.</p>

  <p>Calling <code>img.save()</code> with one argument, as in
  <code>img.save('photo.png')</code>,

  will set the image's filename and type together.</p>

  <p>Calling <code>img.save()</code> with two arguments, as in

  <code>image.save('photo', 'png')</code>, will set the image's filename and
  type

  separately.</p>

  <p>The image will only be downloaded as an animated GIF if the

  <a href="#/p5.Image">p5.Image</a> object was loaded from a GIF file.

  See <a href="#/p5/saveGif">saveGif()</a> to create new GIFs.</p>

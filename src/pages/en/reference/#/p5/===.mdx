  <p>The strict equality operator <a href="#/p5/===">===</a>

  checks to see if two values are equal and of the same type.</p>

  <p>A comparison expression always evaluates to a <a
  href="#/p5/boolean">boolean</a>.</p>

  <p>From <a
  href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Comparison_Operators">the
  MDN entry</a>:

  The non-identity operator returns true if the operands are not equal and/or
  not of the same type.</p>

  <p>Note: In some examples around the web you may see a double-equals-sign

  <a
  href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Comparison_Operators#Equality">==</a>,

  used for comparison instead. This is the non-strict equality operator in
  Javascript.

  This will convert the two values being compared to the same type before
  comparing them.</p>
